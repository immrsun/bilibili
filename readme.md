文件(info.txt)格式要求是：

第一行是视频的地址(地址后面不要跟 "?p=1" 这些字符)

第二行是空白行

其余行是按顺序依次是每个视频的名字(名字随意，有多少个视频就要有多少个 名字 )
(虽然这个链接下有很多视频，写了了两个名字，就只会下载前两个)

--------------------------------------------------------------------------------
